Информация. Методы определения информации. Её фуекции.

Существует два фундаментальных метода по определению, что такое информация:

  • Информация - это сведения, которые живые организмы получают, обрабатывают, запоминают и передают из окружающего мира. Информация может фиксироваться в виде знаний.
  • Информация - это характеристика любого объекта окружающего мира. Эта характеристика складывается из параметров объекта (форма, цвет, матириал и т.д.)

    • Поскольку именно человек работает с информацией, в основном используется первый метод. С этой точки зрения выделяют виды и свойства информации:

  • Зрительная (форма, цвет, расстояние)
  • Звуковая (громкость, тональность)
  • Осязательная (форма, материал, размер)
  • Обонятельная (запах: резкий, приятный - неприятний, силный - слабый)
  • Вкусовая (вкус: сладкий, солёный, кислый, горький)

    • Свойства информации:

  • Понятность
  • Полезнолсть (Полезность каждый организм определяет для себя сам)
  • Достоверность
  • Актуальность

  • Сбор сведений

    • Человек собирает информацию органами чувств, а так же с помощью инструментов. Сбор информации в технике происходит с помощью датчиков.

  • Обработка информации

    • Получение сведений на основе уже имеющихся. В технике информация обраатывается с помощью алгоритмических рассчётов.

    • Хранение осуществляется на внешних носителях. История внешних носителей по времени почти совпадает с историей человечества. Наиболее распространённый способ хранения информации на сегодняшний день - в виде множества копий на электронных носителях, к которым можно получить доступ из сети.

    Кодирование информации.

    Любая информация на компьютере - это набор бит. У разных видов информации кодируются разные минимальные элементы. Минимальный элемент текста - символ. Минимальный элемент картинки - пиксель. У звука - однократное измерение его значения (таких измерений в секунду может быть несколько тысяч).

    Кодирование текстовой информации.

    Количество бит, которое тратится на кодирование одного символа, зависит от количества символов в алыавите. Например, с помощью одного бита можно закодировать два символа. Поскольку в тексте нужно кодировать не только буквы, но и знаки препинания, то первый алфавит состоял из 100 символов. Каждый символ кодировался семью бимтами. Первый алфавит содержал только буквы латиницы, поскольку использовался только на территории США. Список всех букв и их кодов называется кодовой таблицей, или таблицей ASCII.

    В дальнейшем таблица ASCII была модифицирована для компьютеров других стран. Семи бит стало не хватать, и таблица ASCII стала состаять из восьми бит. Существует много локальных вариантов ASCII. Чтобы документы можно было читать в разных странах, на компьютере должны были быть установлены все кодовые таблицы.

    С развитием интернета появилась необходимость читать любые документы в любом месте. Была создана кодировка под названием Unicode. В этой таблице каждый символ кодировался двумя байтами, что позволяло закодировать 65536 символов. Эта таблица включала большее кольчество печатных символов, математических обозначений, множество дополнительных символов. Многие символы пришли из иконографики и позволяли не перегружать интернет-страницы допольнительными рисунками - иконками. На сегодняшний день ведутся разработки по созданию алфавита, состоящего из 36 бит, чтобы поместить все известные символы.
